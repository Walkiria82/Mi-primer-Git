Practicando Git


Comandos Git

  1. Primer comando: git init (Para inmiciargit)
  2. Segundo comando: git status (Para checkear el estado de nuestro git en proyecto)
  3. Tercer comando: git add . (Para trackear todos archivos)
  4. Cuarto comando: git commit -m "mensajito" (Para hacer el paquete de la version que subimos a la nube)
